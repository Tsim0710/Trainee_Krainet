# Trainee_Krainet


## Описание

Этот репозиторий содержит Docker Compose конфигурацию для развертывания GitLab Community Edition (CE) и настройки CI/CD пайплайнов. Также включены инструкции по управлению Docker образами.

- Docker
- Docker Compose
- Docker-Desktop
- Git
